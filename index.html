Get started with sigma

The JavaScript way to draw networks

Sigma makes network exploration easy and even fun to do. It's simple, elegant, and designed so that you spend less time making your hard work and more time designing.

Features

sigmajs is basically a graph model and a controller, that binds renderers to the graph. Currently, it has a Canvas and a WebGL renderers. Also, it is possible for the user to drag and drop the graph and to zoom with the mouse wheel. And has touch supports.

Use and contribute

The source code is tested, commented and documented in the wiki. Also, sigmajs is released under the MIT Licence and published on Github.

Customize

sigmajs provides a lot of different settings to make it easy to customize how to draw and interact with networks. And you can also directly add your own function to your scripts to render nodes and edges the exact way you want.

Resources

The documentation, examples and plugins provided in the core repository bring a lot of different resources to find out how to implement the interactions you want for your applications.
